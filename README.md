# Git Assignment Project

STUDENT: Abid Rafiqzad


This is a **Git & GitHub practical assignment** where I learned how to *initialize a repository*, manage files using `.gitignore`, and use **GitHub for version control and collaboration**.



##  Tools Used for Assignment
- Git
- GitHub
- Visual Studio Code
- Markdown


##  Steps to Initialize Repository
1. Created a project folder
2. Initialized Git using `git init`
3. Created required files
4. Configured `.gitignore`
5. Added files to staging
6. Committed changes
7. Connected to GitHub
8. Pushed code to remote repository


##  File Structure

| File Name   | Purpose                          |
|------------|----------------------------------|
| README.md  | Project documentation            |
| script.js  | JavaScript file for testing      |
| config.env | Stores sensitive environment data|
| .gitignore | Prevents tracking sensitive files|



## Note: 
The `config.env` file is intentionally ignored using `.gitignore` to protect sensitive data.
